# 遵的珠咪

校园猫咪档案小程序，记录校园内猫猫的成长轨迹。

基于 [中大猫谱（笃行猫谱）](https://github.com/Zing22/zhongdamaopu) 二次开发。

## 功能

- **猫猫档案**：校园猫咪资料、相册、精选瀑布流
- **用户互动**：上传照片、留言、点赞、反馈
- **拍照识猫**：拍照识别校园中的猫咪
- **公告系统**：首页公告弹窗，支持频率控制
- **猫际关系**：可视化猫咪关系图谱
- **徽章系统**：用户赠予猫咪徽章
- **榜单**：拍照月榜等功能
- **管理后台**：资料管理、照片审核、公告发布等均在小程序端完成

## 体验

微信搜索或扫码体验"遵的珠咪"小程序。

## 技术栈

- 微信小程序原生框架
- 阿里云 EMAS Serverless（后端/云函数）
- 腾讯云 COS（图片存储）
- MongoDB（EMAS 内置）
